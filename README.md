# Chalet Michael Guest App

Mehrsprachige, installierbare Gäste-Web-App für Chalet Michael.

## Lokal öffnen
Für eine einfache Vorschau `index.html` öffnen. Für die Installations- und Offline-Funktion im Ordner ausführen:

```bash
python3 -m http.server 8000
```

Danach `http://localhost:8000` öffnen.

## Kostenlos über GitHub Pages veröffentlichen
1. Neues Repository `chalet-michael-guest-app` erstellen.
2. Alle Dateien aus diesem Ordner hochladen.
3. **Settings → Pages** öffnen.
4. **Deploy from a branch**, `main`, `/ (root)` auswählen.
5. Speichern; GitHub zeigt danach die öffentliche Adresse an.

## Vor Veröffentlichung ergänzen
- Petras Telefonnummer und WhatsApp-Link in `index.html`
- WLAN-Name und Passwort
- Geräte-, Heizungs-, Abfall- und Check-out-Anleitungen
- Chalet-Fotos und finales Logo
- Genaue Adresse bzw. Kartenlink
